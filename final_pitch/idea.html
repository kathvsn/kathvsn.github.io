Idea  

Using the bullet journal as my inspiration, my idea is that the webpage I'll be creating will act as an actual diary/journal page. Usually one would use one page to log their activities for one specific day, but with bullet journaling, you can fit a week's worth of activities onto 1-2 pages. I want to be able to implement digital aspects that a physical bullet journal doesn't have. Some of these aspects would include the dynamic display of the time and date. Using javascript/jquery, I want to be able to switch to a different page that will include a photo log where I can post a photo with a brief caption and I would also try to implement a settings page where I can change the color of certain elements on the page and the font style.
